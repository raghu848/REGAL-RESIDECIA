# Regal Residencia Website - Network Access Information

## Local Access
- Main URL: http://localhost:3001
- Alternative: http://127.0.0.1:3001

## Network Access
- Network URL: http://192.168.1.23:3001
- Port: 3001

## Instructions
1. Ensure your firewall allows connections on port 3001
2. Other devices on the same network can access the site using the Network URL above
3. Mobile devices can scan the QR code below for easy access
